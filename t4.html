This program fetures calculations, graphing tools, pivot tables, and a macro programming language.

Pre-requisites

  • LibreOffice
  • Hvae a functional Mac

Steps

  1. Go to your applications and Open LibreOffice
  2. Click on "Calc" icon (figure 11)
  3. Input the Information Clicking on the boxes(figure 12)
  4. Click icons on the Formatting tool bar to change your text look

    Like Font size, Font style, tables, and highlight text

  5. Click on the standard tool bar to perfom additional commands

    Save, Print, Paste, and opnem new document

  6. Select the Floppy disk
  7. Choose file name and location
  8. Click "Save"
  9. Close the program
