Downloading LibreOfffice is a simple task. Make sure you meet the system requirements for your Mac OS before downlodaing.

Context

Be sure to have a moderate or higher internet conection.

System requirements

  • Mac OS X 10.6 (Snow Leopard) or higher

  • Intel processor

  • 512 Mb RAM

  • Up to 800 Mb available hard disk space

  • 1024x768 graphic device with 256 colors

Steps

  1. Open your default browser (Internet Explorer, Chrome, Safari, FireFox)

    go in http://www.libreoffice.org/

  2. Click on the top tab "Download" and select LibreOffice Fresh(Figure 2)
  3. Click on the "Download Version 4.3.0" tab

    Make sure to Select the correct operating system "Mac OS"

    (Figure 3)

Post-requisites

  • Close Download window from browser.
